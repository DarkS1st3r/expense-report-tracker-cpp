# expense-report-tracker-cpp
A C++ terminal application for tracking personal finances, generating expense reports, and monitoring budgets.

This project is being developed as part of my software development portfolio while studying Computer Science.

---

## Features

### Current Features
- Add new expenses
- View all expenses
- Save expenses to a file
- Load expenses from a file

### Planned Features
- Edit existing expenses
- Delete expenses
- Search expenses
- Sort expenses by date, category, or amount
- Daily, weekly, monthly, and yearly reports
- Budget tracking
- Spending statistics
- CSV export
- Input validation
- Improved terminal interface

---

## Expense Information

Each expense contains:

- Expense ID
- Date
- Category
- Description
- Amount
- Payment Method

Example:

| Field | Value |
|-------|-------|
| ID | 001 |
| Date | 2026-07-01 |
| Category | Food |
| Description | Grocery Shopping |
| Amount | R450.00 |
| Payment Method | Debit Card |
